## Morning 早晨模式

**字段顺序**：

| 字段 | 问法 | 格式 |
| ---- | ---- | ---- |
| 入睡时间点 | 昨晚几点睡的？ | `HH:MM` |
| 起床时间 | 今早几点起？ | `HH:MM` |
| 起床困觉 | 起床困难吗？(1-5) | 数字 1-5 |
| 前一晚睡眠时长 | （自动计算确认） | `Xh` |
| 能量水平 | 能量状态？(1-5) | 数字 1-5 |
| 专注水平 | 专注力？(1-5) | 数字 1-5 |
| 晨间/昨日 | 早晨有什么想记的？ | 文本 |
| 今日计划 | 今天有什么计划？ | 追加章节 |

**示例**：

```
用户: /daily morning
Claude: 早上好！☀️ 昨晚几点睡的？
用户: 12点半，8点起
Claude: 睡了 7.5h，起床困难 1-5？
用户: 3
Claude: 能量和专注？(1-5)
用户: 4和3
Claude: 今天计划？
用户: 写方案
Claude: ✅ 睡眠 00:30→08:00(7.5h)，能量4专注3，计划：写方案
```

## Evening 晚间模式

**字段顺序**：

| 字段 | 问法 | 格式 |
| ---- | ---- | ---- |
| 今日状态评分 | 今天状态几分？(1-5) | 数字 1-5 |
| 今日发生了 | 今天发生了什么？ | 追加列表 |
| 大奶酪高质量陪伴 | 陪宝宝了吗？ | `true/false` |
| 习惯记录 | 完成了哪些习惯？ | 列表 |
| 今日阅读 | 今天读了什么？ | 文本 |
| 电子娱乐时长 | 电子娱乐多久？ | 分钟数 |
| 印象深刻 | 对什么印象最深？ | 追加段落 |
| 学到什么 | 今天学到什么？ | 追加段落 |
| 感恩/关怀 | 有什么想感恩的？ | 追加列表 |

**示例**：

```
用户: /daily evening
Claude: 晚上好！🌙 今天状态几分？(1-5)
用户: 4分
Claude: 今天发生了什么？
用户: 开会、写代码、陪娃积木
Claude: 陪宝宝了！完成了什么习惯？
用户: 自习
Claude: 电子娱乐多久？
用户: 30分钟
Claude: 印象最深的事？
用户: 老板表扬方案
Claude: 有什么想感恩的？
用户: 团队配合
Claude: ✅ 状态4，事件3项，陪娃✓，习惯：自习，娱乐30min
```

## Quick 快速模式

批量收集，一次问完。

**早晨快速**：

```
Claude: 早！☀️ 睡眠(几点睡-起)、能量/专注(1-5)、今日计划？
用户: 12点睡8点起，能量4专注3，写方案
Claude: ✅ 记录完成
```

**晚间快速**：

```
Claude: 晚！🌙 状态(1-5)、今天发生啥、陪娃了吗、感恩？
用户: 4分，开会写代码陪娃，陪了，感谢团队
Claude: ✅ 记录完成
```

## Deep 深度模式

每个回答后追问一层。

**追问示例**：

| 基础问题 | 追问 |
| -------- | ---- |
| 昨晚几点睡？ | 入睡顺利吗？ |
| 今天状态几分？ | 是什么让你给这个分数？ |
| 对什么印象深刻？ | 为什么这件事触动你？ |
| 今天学到什么？ | 这会如何影响你接下来的行动？ |
| 有什么想感恩的？ | 这份感激想如何表达？ |

**示例**：

```
用户: /daily deep
Claude: 晚上好 🌙 今天状态几分？
用户: 4分
Claude: 是什么让你给 4 分？
用户: 工作顺利但有点累
Claude: 累是身体上的还是心理上的？
用户: 开会太多，脑子转不动
Claude: 今天发生了什么？
...
```
